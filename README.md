# Exercícios de Lógica de Programação

Este repositório contém uma coleção de **exercícios de lógica de programação**, desenvolvidos para praticar e aprimorar habilidades em **Java** estrutura de dados e algoritmos.

## Objetivo

- Praticar **resolução de problemas** e **pensamento lógico**.  
- Treinar a implementação de algoritmos em **Java**.  
- Aprender a estruturar soluções e organizar estruturas e pensamento crítico.
