# Queue
Primeiro a entrar é o primeiro a sair. 

Analogia vida real : fila de banco. 

Ideias para processamento sequencias: tarefas, requisições, impressão etc

## Operacoes 
- offer : Add ao final da fila 
- poll: remove primeiro elemento 
- peek: visualiza o primeiro sem remover

## Priority Queue 
É uma fila que ordena automaticamente os 
elementos conforme a prioridade e é baseada em Heap
