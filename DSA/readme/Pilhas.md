# Pilha / Stacks
segue a lógica LIFO : o ultimo a entrar é o primeiro a sair.
pense como se fosse pilha de pratos a ser lavados.
usamos stacks para navegacao entre telas, desfazer acoes e analisar expressoes matematicas

## Operaçoes
push : add no topo 
pop :  delete do topo 
peek : visualiza o top sem mexer
